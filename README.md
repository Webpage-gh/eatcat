### 键型设置文档

概述
本文档介绍如何设置自定义键型，以控制游戏中按键的生成位置。键型由一系列字符组成，每个字符代表特定的生成规则。

基本符号说明

符号	说明
`!`	在随机位置生成按键
`@`	在与上一按键不同的位置随机生成
`#`	在与上一按键相同的位置生成
`&`	在上一按键的对称位置生成
`+n`	在上一按键位置向右移动n个轨道生成
`-n`	在上一按键位置向左移动n个轨道生成
`%mn`	在轨道m到n之间随机生成
`*nabc...`	在指定的n个轨道中随机选择其中一个生成
`1-9`	在指定轨道生成（轨道编号从1开始）

使用示例

示例1：基础随机

```
!@!#
```

表示：随机 → 不同位置 → 随机 → 相同位置

示例2：对称移动

```
1&2&3&
```

表示：在轨道1 → 对称位置 → 轨道2 → 对称位置 → 轨道3 → 对称位置

示例3：指定范围随机

```
%13!%24
```

表示：在轨道1-3随机 → 完全随机 → 在轨道2-4随机

示例4：指定轨道选择

```
*3123!*42456
```

表示：在轨道1/2/3中随机选择 → 完全随机 → 在轨道2/4/5/6中随机选择

注意事项
1. 轨道编号从1开始，最大轨道数由当前设置决定
2. 使用`+n`/`-n`时需注意边界，会自动循环
3. 多个符号可以组合使用形成复杂模式
4. 空键型默认为`!`

高级技巧
- 使用对称模式`&`可以创建镜像效果
- 结合范围随机`%`和指定选择`*`可以创建局部随机效果
- 通过`+`/`-`可以创建单向移动模式
